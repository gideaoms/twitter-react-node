# Projeto Twitter

## Stacks

- NodeJS
- ReactJS
- Twitter Api

## Como usar

- entrar na pasta backend
- executar npm install
- executar npm run dev

- entrar na pasta frontend
- executar npm install
- executar npm run dev

- visitar a url http://localhost:8080

## Testes

- entrar na pasta backend
- executar npm test

- entrar na pasta frontend
- executar npm test
